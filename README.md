# Codex Token Usage

本项目用于在本机查看和分析 Codex 会话日志中的 token 消耗，帮助用户按明细和统计两个视角理解 token 用量。

## 1. 项目简介

Codex Token Usage 是一个面向个人本机的 Codex token 观察工具。

它读取本机 Codex 日志，展示每条消息、turn 和 token_count 的消耗明细，并按项目、会话和时间维度生成统计结果。

## 2. 主要功能

1. 查看明细：按 `Project -> Session -> 用户输入 -> Turn -> TokenCount` 展开 token 用量，默认只显示项目层级，并支持按层级按钮批量展开到会话、用户输入、Turn 或 TokenCount。
2. 查看统计：默认进入统计页，展示半年趋势、近 2 周趋势、近 2 周分时、Token 构成、项目排行和会话排行。
3. 筛选分析：按日期范围、项目、会话名称和输入内容查询记录；选择项目后会话列表联动收窄，日期范围默认近 14 天。
4. 配置设置：支持在工具栏直接编辑会话目录和定时刷新，归档会话默认纳入统计。
5. 导出结果：支持在明细页导出明细 CSV，默认只包含输入预览。
6. 数据维护：支持导出结果和安全重建数据库，重建前会自动备份本工具 SQLite 数据库。
7. 桌面常驻：提供托盘入口，可打开窗口、刷新统计、检查更新和退出。
8. 版本更新：使用内置 GitHub Latest Release 更新源，支持检查到新版本后一键更新稳定入口。

功能描述应从用户视角编写，不使用开发术语。

## 3. 使用前准备

1. 本机已经使用过 Codex，并存在本地会话日志。
2. 工具只能统计本机可读取的日志，不代表官方账单或官方额度口径。
3. 当前项目提供本机桌面开发版，可读取当前电脑上的 Codex 会话日志并生成本地账本。

## 5. 使用说明

1. 启动桌面工具。
2. 点击刷新，读取本机 Codex 会话日志。
3. 在统计页查看趋势、分时分布、Token 构成、项目排行和会话排行。
4. 切换到明细页查看逐层展开的 token 记录。
5. 通过日期范围、项目、会话和关键词查询记录，再在明细页按需切换仅异常显示。
6. 在明细页导出明细 CSV，或在工具栏中按需重建本工具数据库。
7. 如需更新工具，在工具栏或托盘菜单点击检查更新，检查到新版本后点击立即更新。
8. 稳定入口会默认创建在 `%LOCALAPPDATA%\Programs\CodexTokenUsage\CodexTokenUsage.exe`，后续更新会替换该入口。

## 6. 注意事项

1. 本工具只读取本机日志，不读取其他人的电脑数据。
2. 本工具统计的是本地日志中的 token 信息，不保证与官方计费或额度完全一致。
3. 本地账本默认保存到 `%LOCALAPPDATA%\CodexTokenUsage\codex-token-usage.sqlite`。
4. 重建数据库操作会先要求确认并自动备份当前 SQLite；这些操作只影响本工具 SQLite 数据库，不删除 Codex 原始日志。
5. 开发工程位于 `app/`；根目录可保留当前构建出的 portable exe。
6. 运行时数据和导出报表默认不进入版本控制。
7. 检查更新失败不影响统计主功能。
8. 外部 updater 只替换稳定入口，不直接覆盖正在运行的根目录 portable exe。

## 7. 常见问题

### 统计结果是否等于官方扣费？

不等于。它是基于本机 Codex 日志的本地统计，主要用于趋势分析和异常排查。

### 会不会看到其他共用账号成员的内容？

不会。工具只读取当前电脑上的本机日志。
